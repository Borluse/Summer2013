Camera viewCamera
 The camera this button is meant to be viewed from. Set this explicitly for best performance.
The system will automatically traverse up the hierarchy to find a camera if this is not set.
If nothing is found, it will fall back to the active tk2dCamera.
Failing that, it will use Camera.main.
